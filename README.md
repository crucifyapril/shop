#### Сборка и запуск проекта:
- Сборка `make build vendor`  
- Запуск `make up`
- Перейти на [localhost](http://localhost)

#### Окружение:
- Laravel 11
- PHP 8.3.11
- MySQL
- Nginx
- Redis
- Docker
